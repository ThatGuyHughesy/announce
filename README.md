## Announce

Growl-inspired alerts and message.


## Install

__npm__
```sh
$ npm install
```

__bower__
```sh
$ bower install
```

## Usage

```javascript
let announce = new Announce({
    timeout: 5000
});

announce.post({
  text: 'This is an announcement!'
});

```

[API documentation](https://thatguyhughesy.github.io/announce/docs)

[Demo](https://thatguyhughesy.github.io/announce/)
